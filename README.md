# React-Components

This is my personal React Styled-Component Library

**To Install:**

```bash
npm install @twitocode/react-components

or

yarn add @twitocode/react-components
```

**Example:**

```typescriptreact
import { Text } from '@twitocode/react-components';

export const App = () => {
  return (
    <>
      <Text color='black' align='center'>
        Hello World!
      </Text>
    </>
  )
}
```
